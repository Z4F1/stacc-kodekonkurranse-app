About

This is just a programming challenge by Stacc AS where we had to create something either with the OpenSea API or something along the lines of NFTs atleast. So here is my app, it uses a backend I've created with Node.js and Express. The app is made with Ionic and Angular (which is built in the Ionic Framework).

In this app, you can bid on items from the OpenSea API which is then sent to the backend and stored with MongoDB. At the bottom of each Asset Page, you can see all the bids on that particular item. If it is your bid, it will say so.
